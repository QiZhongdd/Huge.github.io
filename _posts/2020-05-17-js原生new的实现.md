# new 的实现原理

- 新建一个空对象。
- 空对象的*proto*指向构造函数的 prototype.
- 传入相应的参数执行函数并改变构造函数的指向。
- 返回这个新对象

```
function new(Fn){
  var obj={};
  obj._proto_=Fn.prototype;
  let args=Array.prototype.slice.call(arguments,1);
  Fn.apply(obj,args);
  return obj
}
```
